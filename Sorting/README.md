# Sorting Algorithms

This folder contains implementations of different sorting algorithms in Java.

## Algorithms Included
- **Selection Sort**  
  - Time Complexity: O(n^2)  
  - Space Complexity: O(1)  
  - Stable: No  

- **Bubble Sort**  
  - Time Complexity: O(n^2)  
  - Space Complexity: O(1)  
  - Stable: Yes
